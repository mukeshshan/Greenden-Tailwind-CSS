🌿 Greenden - Plant Shop Website

Greenden is a simple and responsive plant shop website built using HTML and Tailwind CSS.
This project focuses on creating a modern UI with a clean layout for an online plant store.

🚀 Features

Responsive design using Tailwind CSS

Clean and modern user interface

Multiple pages for easy navigation

Simple layout suitable for beginners learning frontend development

📄 Pages Included

Home Page

Introduction to Greenden

Featured plants

Promotional sections

Products Page

Display of different plant products

Product cards with images and details

Contact Page

Contact form

Basic user information inputs

🛠️ Built With

HTML5

Tailwind CSS
